> 如何把Visual Studio Code设置成中文

##方法/步骤
### 1.打开vscode,快捷键 Ctrl+Shift+X,或者点击图标打开插件商店

### 2.在搜索框中输入Chinese,找到Chinese (Simplified) Language Pack for Visual Studio Code（一般是第一个），点击安装

### 3.右下角会有弹框出现，点yes，重启vscode，完成
